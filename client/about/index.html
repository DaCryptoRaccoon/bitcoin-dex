Bitcoin DEX

A Decentralized Bitcoin Exchange

Main Features

  • Non-custodial BTC/USDt trading.
  • Native to Bitcoin. Real BTC. There's no counterparty risk of bitcoin derivatives.
  • Simple, installation-free web wallet.

Limitations

  • This is an experimental prototype. We tested it thouroughly, we're watching it carefully, but still, things might go wrong.
  • Currently, we use wrapped Tether called "USDx". This is a temporary workaround until we've finished implementing a trustless 2-way peg between Ethereum and Bitcoin.
